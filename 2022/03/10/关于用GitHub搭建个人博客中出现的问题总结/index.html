关于用GitHub搭建个人博客中出现的问题总结

Created2022-03-10Updated2022-04-15

通过自己的努力，实现了利用Hexo搭建个人博客，下面总结搭建过程中出现的问题。

1.初始化个人博客

使用命令：hexo init时，我出现了如下的错误（我分析应该是权限不够的问题）

解决办法：关闭cmd，在搜索栏里右键命令提示符，点击以管理员身份运行，重新进入到本地博客存放目录，输入hexo init，应该就可以了。如下方界面：
![X_O`MC1P_4K@_Y_R_MLLJ_2.png](https://s2.loli.net/2022/04/15/1UL8QAms7vOEcgH.png)

2.发布到github

命令：hexo d
我出现了一个错误

这是因为没安装hexo-deployer-git插件，在站点目录下输入下面的插件安装命令：
npm install hexo-deployer-git --save

然后再使用hexo d命令就可以推送了。
