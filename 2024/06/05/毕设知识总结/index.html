使用 dayjs 库进行格式化

  • time = this.dayjs(time).format('YYYY-MM-DD HH:mm:ss');
    • this.dayjs(time)：假设 dayjs 是一个已经在当前上下文中引用的库（通常在 Vue 组件中使用 this 引用库函数），通过 dayjs 将传入的 time 转换为一个 dayjs 对象。
    • .format('YYYY-MM-DD HH:mm:ss')：使用 dayjs 对象的 format 方法将日期时间格式化为指定的字符串格式
    • YYYY-MM-DD HH:mm:ss，其中：
      • YYYY：表示四位数的年份。
      • MM：表示两位数的月份。
      • DD：表示两位数的日期。
      • HH：表示两位数的小时（24小时制）。
      • mm：表示两位数的分钟。
      • ss：表示两位数的秒钟。

返回格式化后的时间字符串

  • return time;：返回格式化后的时间字符串。

这样，你就可以在 Vue 组件中使用 timeChange 方法来格式化日期时间数据。
